# ADR-0002

Status: ACCEPTED

## Open Source First Policy

AurumAI never rebuilds mature open-source components.

Priority:

1. Reuse
2. Extend
3. Integrate
4. Build only if unavailable

Applies to:

- Technical Indicators
- Backtesting
- Collectors
- Knowledge Graphs
- APIs
- Risk Models
- Time Series
- Portfolio Optimization
